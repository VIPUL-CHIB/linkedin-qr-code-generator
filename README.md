# 🔗 LinkedIn QR Code Generator

This project demonstrates how to generate a customized QR code using Python.  
The QR code links directly to a LinkedIn profile and is styled with a red foreground and white background.

---

## 📌 Features
- Generate QR codes with the [`qrcode`](https://pypi.org/project/qrcode/) library.
- Customize QR code appearance (colors, box size, border).
- Save QR codes as image files (`.png`).
- High error correction level (H) for better reliability.

---

## 🛠️ Requirements
Make sure you have the following installed:

- Python 3.x
- [qrcode](https://pypi.org/project/qrcode/)  
- [Pillow (PIL)](https://pypi.org/project/Pillow/)

Install dependencies with:
```bash
pip install qrcode[pil] Pillow
